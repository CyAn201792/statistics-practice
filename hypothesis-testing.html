Hypothesis Testing

Hypothesis testing is a statistical method used to determine whether there is enough evidence to reject a null hypothesis based on sample data.

Key Concepts

  • Null hypothesis (H₀)
  • Alternative hypothesis (H₁)
  • p-values
  • Type I and Type II errors
  • Significance level (α)

Common Tests

  • Z-test
  • T-test
  • Chi-square test
